Introduccion:

En la actualidad es de suma importancia saber acerca de todo lo nuevo en tecnología ya que el que no sabe ni tan siquiera usar una computadora se está quedando muy atrás. En este parcial se llevaron a cabo de varias practicas sobre las cuales se aprendieron algunas cuantas etiquetas en HTML. Antes de empezar comenzaremos explicando que es HTML y sus etiquetas básicas. HTML, es el lenguaje estándar utilizado para crear y diseñar páginas web. Es fundamental en la estructura de cualquier sitio en la web, ya que define la forma en que se presentan los contenidos en un navegador. Las etiquetas HTML son elementos clave que permiten organizar y dar formato a la información en una página web. Estas etiquetas básicas son fundamentales para comenzar a construir una página web con HTML. definiendo la disposición y relación de los elementos que la componen. Al combinar HTML con otros lenguajes como CSS para el diseño y JavaScript para la interactividad, se pueden crear sitios web dinámicos, atractivos y funcionales Y a continuación se presentarán las siguientes practicas donde se llevaron a cabo algunas de estas etiquetas

Para entender el funcionamiento básico de HTML5 es importante conocer los fundamentos de los lenguajes de marcas y su evolución desde sus primeras versiones allá por los años 60 ya que nos va a permitir introducirnos en el mundo de la creación de páginas web. HTML5 simplifica y mejora la creación de documentos HTML, eliminando etiquetas obsoletas, apoyándose en las hojas de estilo para mejorar su aspecto, en JavaScript para ganar dinamismo y funcionalidad y en sus nuevos elementos para no tener que usar programas externos o para crear una estructura más coherente dentro de una página web. Ante las innovaciones que aporta HTML5 es interesante saber el grado de compatibilidad que tienen los navegadores a la hora de tratar con los nuevos elementos HTML5 ya que esto nos permitirá utilizar esos elementos para que no haya problema a la hora de verlos en cualquier navegador. También hay que ver que HTML5 va más allá de lo que es un lenguaje de marcas y verlo como una agrupación de especificaciones que están marcando el desarrollo web. En definitiva, tal y como lo describe el consorcio W3C, HTML5 es el futuro del contenido.
